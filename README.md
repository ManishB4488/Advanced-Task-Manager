Advanced-Task-Manager
A fully responsive task management web app built with HTML, CSS, and JavaScript.
Features
- Add / Edit / Delete tasks
- Categories
- Due dates
- Priority levels
- Search & Filter
- LocalStorage persistence
- Dark / Light mode
- Fully responsive design
Tech Stack
- HTML5
- CSS3
- JavaScript
